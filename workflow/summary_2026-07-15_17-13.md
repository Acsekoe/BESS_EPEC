## Current objective

Continue the spot-market BESS EPEC with scarce nodal BESS power access. The immediate issue is that diagonalization update rules allocate scarce nodal access differently, so the model needs a more defensible access-allocation mechanism than pure Jacobi projection or Seidel residual capacity.

- Both update rules identify N1 and N5 as the valuable nodes in the new Euro data.

- Therefore, the exact 50/50 Jacobi split should not be reported as a unique market prediction. It is a symmetric rationing outcome induced by the update/projection rule.

## Proposed next modeling step: nodal BESS power-access auction

The current shared access constraint is:

```text
sum_i X_power[i,n] <= Limit[n]
```

Jacobi plus projection implements a neutral but artificial proportional rationing rule. Seidel implements a sequential residual-capacity rule and can create first-mover advantage. A more defensible mechanism is an explicit nodal BESS power-access auction.

Suggested formulation:

- each node `n` has `Limit[n]` MW of BESS grid connection capacity
- investors demand/access `X_power[i,n]`
- a nodal access price `kappa[n] >= 0` clears scarce connection capacity
- investor objective subtracts an access payment:

```text
profit_i =
spot revenue
- degradation cost
- CAPEX
- sum_n kappa[n] * X_power[i,n]
```

Access-market clearing:

```text
sum_i X_power[i,n] <= Limit[n]
kappa[n] >= 0
kappa[n] * (Limit[n] - sum_i X_power[i,n]) = 0
```

Pragmatic implementation path:

1. Add `access_price_eur_per_mw_day[n]` or `kappa[n]` as an exogenous parameter to each investor MPEC.
2. Subtract `sum_n kappa[n] * X_power[n]` from `investor_profit_expr`.
3. In the EPEC outer loop, update `kappa[n]` with a tatonnement-style rule:

```text
kappa[n] = max(0, kappa[n] + step * (sum_i X_power[i,n] - Limit[n]))
```

4. Remove or reduce reliance on proportional projection; use projection only as a numerical guard while the auction price converges.
5. Converge on both capacities and access excess demand.
6. Report `kappa[n]` as the market-clearing value of scarce BESS connection access.

Research narrative:

> Jacobi and Seidel show that naive diagonalization makes scarce nodal access allocation algorithm-dependent. A nodal access auction replaces solver-order rationing with an explicit price signal for scarce BESS grid connection capacity.
